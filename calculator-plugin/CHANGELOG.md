# 更新日志

## V1.0.0 (2026-06-09)

### 首次发布

高级计算器 uTools 插件正式发布，完整移植桌面版全部功能。

### 功能模块

- **基础计算器** — 四则运算、百分比、正负切换、键盘快捷键
- **科学计算器** — 三角函数、对数、指数、根号、阶乘、取整、常数π/e
- **方程求解** — 一元一次、一元二次（复数解）、二元一次方程组
- **矩阵运算** — 加减乘（2×2）、行列式与逆矩阵（3×3）
- **几何计算** — 10种图形：圆、三角形、矩形、椭圆、梯形、正多边形、球体、圆柱体、圆锥体、圆环体
- **概率统计** — 组合排列、描述统计、二项/正态/泊松分布
- **数论工具** — 素数检测、素因数分解、GCD/LCM、模幂、欧拉函数、斐波那契
- **单位转换** — 8大类：长度、重量、温度、面积、体积、数据、速度、进制
- **趣味工具** — 房贷车贷、年化收益、汇率、大写数字、进制、速度、BMI、个税
- **数学教程** — 6大主题交互式学习

### 其他特性

- 支持中文、English、हिन्दी、Español 四种语言
- 纯前端实现，无需后端服务
- 所有计算本地完成，无数据上传
- 模块化架构，10个独立JS模块
