高级计算器 - 新功能说明
========================

已成功添加四个新功能：

1. 矩阵计算器 (Matrix Calculator)
-------------------------------
功能：
- 矩阵加法 (2×2矩阵)
- 矩阵减法 (2×2矩阵)
- 矩阵乘法 (2×2矩阵)
- 行列式计算 (3×3矩阵)
- 逆矩阵计算 (3×3矩阵)

使用方法：
1. 点击"矩阵"标签页
2. 选择运算类型（加法、减法、乘法、行列式、逆矩阵）
3. 输入矩阵元素
4. 点击"计算"按钮

示例：
- 加法: [[1,2],[3,4]] + [[5,6],[7,8]] = [[6,8],[10,12]]
- 行列式: det([[1,2,3],[4,5,6],[7,8,10]]) = -3

2. 几何工具 (Geometry Tools)
---------------------------
功能：
- 圆: 面积、周长
- 三角形: 面积、周长（海伦公式）
- 矩形: 面积、周长
- 球体: 体积、表面积
- 圆柱体: 体积、侧面积、总表面积

使用方法：
1. 点击"几何"标签页
2. 选择几何图形
3. 输入参数（半径、边长等）
4. 点击"计算"按钮

示例：
- 圆: 半径=5 → 面积=78.54, 周长=31.42
- 三角形: 边长=3,4,5 → 面积=6, 周长=12

3. 3D计算器 (3D Calculator)
--------------------------
功能：
- 向量运算: 点积、叉积、模长、夹角
- 两点距离: 3D空间中两点间距离
- 平面方程: 点到平面距离
- 直线方程: 参数方程表示

使用方法：
1. 点击"3D"标签页
2. 选择运算类型
3. 输入坐标或向量分量
4. 点击"计算"按钮

示例：
- 向量点积: (1,2,3)·(4,5,6) = 32
- 两点距离: (1,2,3)到(4,5,6)的距离 = 5.196

技术实现：
--------
- 纯JavaScript实现，无外部依赖
- 支持键盘快捷键
- 暗色主题，护眼舒适
- 响应式设计

运行方式：
--------
1. 双击 electron-calculator\run-and-test.bat
2. 或者双击 electron-calculator\run.bat
3. 等待依赖安装完成
4. 计算器将自动启动

构建可执行文件：
--------------
1. 双击 electron-calculator\build-win.bat
2. 等待构建完成
3. 可执行文件将生成在 electron-calculator\dist 目录中

4. 个人所得税计算器
--------------------------
功能：
- 根据中国大陆个人所得税法计算
- 七级超额累进税率（3%-45%）
- 支持五险一金扣除
- 支持专项附加扣除
- 显示应纳税额、实发工资、年缴税额

使用方法：
1. 点击"趣味"标签页
2. 点击"个税计算"按钮
3. 输入月收入、五险一金、专项附加扣除等信息
4. 点击"计算"按钮

示例：
- 月收入15000元，五险一金3000元，起征点5000元
- 应纳税所得额 = 15000 - 3000 - 5000 = 7000元
- 适用税率10%，速算扣除210元
- 应缴个税 = 7000 × 10% - 210 = 490元/月

5. 微积分计算器 (参考 MathWorld)
--------------------------
功能：
- 导数: 数值求导(中心差分法)，支持任意阶导数
- 积分: 定积分(辛普森法/梯形法)，参考 Wolfram 积分计算器
- 极限: 数值极限计算，左/右/双侧极限
- 级数: 任意通项级数求和，展示著名级数公式
- 泰勒展开: 数值泰勒展开，展示常用展开公式

使用方法：
1. 点击"微积分"标签页
2. 选择运算类型（导数/积分/极限/级数/泰勒展开）
3. 输入函数表达式和参数
4. 点击"计算"按钮

6. 概率统计计算器 (参考 MathWorld)
-----------------------------
功能：
- 排列组合: P(n,r)、C(n,r)
- 描述统计: 均值、中位数、众数、方差、标准差、极差
- 二项分布: PMF、CDF、期望、方差
- 正态分布: PDF、CDF、标准正态Z值
- 泊松分布: PMF、CDF

7. 数论计算器 (参考 MathWorld)
--------------------------
功能：
- 素数检测: Miller-Rabin确定性检测
- 素因数分解: 完整素因数列表
- GCD/LCM: 最大公约数/最小公倍数
- 模幂运算: 快速模幂算法
- 欧拉函数: φ(n)计算
- 斐波那契: 快速计算F(n)

8. 代数扩展计算器 (参考 MathWorld)
------------------------------
功能：
- 三次方程: Cardano公式求解
- 多项式: 霍纳法求值
- 集合运算: 并集、交集、差集
- 逻辑运算: 与、或、非、蕴含、等价
- 数列求和: 等差/等比数列

9. 应用数学计算器 (参考 MathWorld)
------------------------------
功能：
- 牛顿迭代法: 求解非线性方程
- 线性回归: 最小二乘法拟合
- 拉格朗日插值: 多项式插值
- 数值积分: 辛普森法
- 微分方程: 龙格-库塔法(RK4)

所有新功能公式和计算方法均参考自 https://mathworld.net.cn/ (Wolfram MathWorld 中文镜像)。
