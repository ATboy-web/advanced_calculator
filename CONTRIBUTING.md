# Contributing to Advanced Calculator

Thank you for your interest in contributing to Advanced Calculator!

## How to Contribute

### Reporting Bugs

1. Check existing [issues](https://github.com/ATboy-web/advanced_calculator/issues) to avoid duplicates
2. Use the [Bug Report](https://github.com/ATboy-web/advanced_calculator/issues/new?template=bug_report.yml) template
3. Include detailed steps to reproduce

### Suggesting Features

1. Use the [Feature Request](https://github.com/ATboy-web/advanced_calculator/issues/new?template=feature_request.yml) template
2. Explain the problem and proposed solution

### Submitting Code

1. Fork the repository
2. Create a feature branch: `git checkout -b feature/your-feature`
3. Make your changes
4. Test thoroughly
5. Commit: `git commit -m "feat: add your feature"`
6. Push: `git push origin feature/your-feature`
7. Open a Pull Request

## Development Setup

### Electron App
```bash
cd electron-calculator
npm install
npm start
```

### Android APK
```bash
cd android-apk
./gradlew assembleDebug
```

## Code Style

- Use consistent indentation (2 spaces for HTML/CSS/JS)
- Add comments for complex logic
- Keep functions focused and small
- Use meaningful variable names

## Commit Convention

Use conventional commits:
- `feat:` new feature
- `fix:` bug fix
- `docs:` documentation
- `style:` formatting
- `refactor:` code restructuring
- `test:` adding tests
- `chore:` maintenance

## Questions?

Open an issue for any questions!
